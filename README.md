# web-codegen-scorer playground

This is a lab.
Please refer to https://github.com/angular/web-codegen-scorer for further details about how to use it.

## Usage

Currently, this environment works with `claude-opus-4.5-no-thinking`,
You need to have an `ANTHROPIC_API_KEY`.

```bash
# provide API Key
export ANTHROPIC_API_KEY="sk-ant..."

# Run evaluation
npm run eval.opus

# Study report
npm run report
```
